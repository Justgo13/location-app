# location-app

This website allows users to create an account and add their favourite places. Users can then edit or delete the places added as long as they own them

Technologies used:

- ReactJS 
  - React Hooks
- MongoDB 
  - Mongoose 
- Express
- Node.js
## Setting up frontend

```
cd location-app-frontend
npm install
npm start
```

## Setting up backend
```
cd location-app-frontend
npm install
npm start
```
